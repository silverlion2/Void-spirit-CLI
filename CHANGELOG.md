# Changelog

All notable changes to Void Spirit will be documented in this file.

## [1.0.0] - 2026-03-21

### Features

- 🔌 **Multi-provider support**: OpenAI, Gemini, Anthropic, DeepSeek, Groq, Ollama, Together AI, custom endpoints
- 🛠️ **12 built-in tools**: File ops, shell commands, git, web fetch, search
- 💾 **Session persistence**: Auto-save on exit, resume with `--resume`
- 🌿 **Conversation branching**: Fork/switch between conversation checkpoints
- ⚡ **Parallel tool execution**: Read-only tools run concurrently
- 📝 **Streaming markdown**: Rich formatted output in terminal
- 🧠 **Memory system**: Persistent cross-session memories
- 🔒 **Security sandbox**: Path sandboxing, command blocklist, audit log
- 🔌 **Plugin system**: Install plugins from GitHub
- 📊 **Token tracking**: Real-time usage and cost estimation
- 🎯 **Project detection**: Auto-detects project type and adapts context
- ↩️ **Undo/rollback**: Revert file changes with `/undo`
